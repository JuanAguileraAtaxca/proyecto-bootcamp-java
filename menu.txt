Uso: java -jar rest-client.jar <comando> [opciones]

Comandos disponibles:
  get       Realiza una solicitud GET para recuperar datos.
            Uso: get <URL>
            Ejemplo: get https://api.example.com/resource

  post      Realiza una solicitud POST para enviar datos.
            Uso: post <URL> <datos>
            Ejemplo: post https://api.example.com/resource '{"name":"John", "age":30}'

  put       Realiza una solicitud PUT para actualizar datos.
            Uso: put <URL> <datos>
            Ejemplo: put https://api.example.com/resource/1 '{"name":"John", "age":31}'

  delete    Realiza una solicitud DELETE para eliminar datos.
            Uso: delete <URL>
            Ejemplo: delete https://api.example.com/resource/1

Opciones adicionales:
  -h, --help        Muestra este mensaje de ayuda.
  -H, --header      Especifica un encabezado de solicitud.
                   Uso: <comando> -H "HeaderName: HeaderValue" <URL> [datos]
                   Ejemplo: get -H "Authorization: Bearer token" https://api.example.com/resource

  -v, --verbose     Muestra información detallada de la solicitud y la respuesta.
                   Uso: <comando> -v <URL> [datos]
                   Ejemplo: post -v https://api.example.com/resource '{"name":"John", "age":30}'

  -f, --file        Envía datos desde un archivo.
                   Uso: <comando> -f <URL> <ruta_al_archivo>
                   Ejemplo: post -f https://api.example.com/resource ./data.json
